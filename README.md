### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8或jdk17`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 车辆展示、5. 用户交流、6. 汽车资讯、7. 留言反馈、8. 个人中心、9. 购物车、10. 在线客服、11. 后台管理

管理员功能：1. 登录、2. 个人中心、3. 用户管理、4. 车辆展示管理、5. 车辆品牌管理、6. 车辆型号管理、7. 维修材料管理、8. 材料分类管理、9. 用户交流、10. 留言板管理、11. 系统管理、12. 订单管理
